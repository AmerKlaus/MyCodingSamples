Amer's Shop Website:

This Shopping Website is a modern and user-friendly e-commerce platform designed to provide users with a 
seamless online shopping experience. The website offers a variety of high-quality tech products, including laptops, 
desktops, and accessories, catering to tech enthusiasts who prioritize product quality.

Features:

Product Listings: Browse through a wide range of tech products, including laptops, desktops, and accessories.

Product Details: Access detailed information about each product, including specifications, prices, 
and detailed review videos.

Shopping Cart: Easily add products to the shopping cart for a convenient and organized shopping experience.

Cart Management: Adjust the quantity of products in the cart, and remove items as needed.

Total Cost Calculation: The website dynamically calculates the total cost of items in the cart, providing 
users with a clear understanding of their expenses.

Order Placement: Users can place orders directly through the website, making the shopping process efficient 
and straightforward.

Technologies Used:

HTML, CSS, JavaScript: The core technologies for building the website's frontend, ensuring a responsive 
and interactive user interface.

Session Storage: Used to store and retrieve data related to the shopping cart and user interactions.

Responsive Design: The website is designed to be accessible and functional across various devices and screen sizes.

Node.js: Used for server-side JavaScript.

npm: The package manager for Node.js. Used to install and manage project dependencies.

json-server: A lightweight JSON file-based REST API server. Utilized to simulate a backend server for the shopping website.

API Calls: Leveraged json-server to create a mock API for handling product data retrieval and other CRUD operations.

Usage:

Open the website in your preferred web browser.

Explore the product listings and click on individual product's image for detailed information.

Add desired items to the shopping cart.

Manage the shopping cart by adjusting quantities or removing items.

Proceed to checkout to place an order.
